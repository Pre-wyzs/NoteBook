# vim使用优化

编写此文档的目的：精益求精！

# 1、行首行尾插入

- 虽然使用大写a和大写i可以分别实现在行首和行尾的插入，但是，要按CapsLk切换大小写，容易把大小写搞混掉就很麻烦，

- **<font color='red'>推荐使用 shift + a 和 shift + i实现行首行尾的插入！（一个可以减少按键次数，一个防止把大小写搞混）</font>**

# 2、行选中与取消

- 之前也是使用V来选中和取消选中的，但是还是那两个问题
- **<font color='red'>推荐使用 shift + v 来进行行选中和取消选中的操作</font>**



**==<font color='red'>注意：就是你选中 key down shift键的时候，中英文是不会切换的，只有key up的时候才会切换！所以使用shift + 命令按键的形式不容易把中英文搞混的！</font>==**



# 3、查找变量/函数出现过的位置

**==<font color='blue'>使用shift + \*锁定，按n往下查找;按N往上查找！如果这时候你不小心按了其他键，导致光标丢失了，没有关系！再按n就能回到目标上了！</font>==**



# 4、大文件翻页功能

**==项目中一个文件就有34百行，使用 翻页功能很方便！ctrl + u向上翻页；ctrl + d向下翻页==**





# 5、跳转匹配括号

**==光标选中一个括号，shift + %匹配括号==**



# 6、查找与跳转指定行

**<font color='blue'>/ 输入查找内容 enter就可以查找指定的内容了。</font>**

**<font color='purple'>: 输入行号，就可以跳转到指定的行了。</font>**
